# E-commerce App (Vanilla JS)

## Overview
This is a simple e-commerce application built using HTML, CSS, and JavaScript.

## Features
- Product listing
- Add to cart
- Cart display
- Data persistence using localStorage

## Note
While building this project, I initially attempted it independently but encountered challenges with structuring the application and managing state across pages. I used external references, including AI assistance, to better understand the architecture and complete the project.

This helped me identify areas I need to improve, especially in building larger applications without guided tutorials.
